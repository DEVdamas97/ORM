# CRUD de Produtos com SQLAlchemy

## Sobre o projeto

Este projeto é um exemplo simples de **CRUD utilizando Python, SQLite e SQLAlchemy (ORM)**.

Ele demonstra como uma aplicação pode **criar, listar, atualizar e deletar dados** em um banco de dados utilizando **classes Python como representação das tabelas**.

O objetivo é entender os conceitos principais de **ORM e manipulação de banco de dados no back-end**.

---

## Tecnologias

- Python  
- SQLite  
- SQLAlchemy  

---

## Conceitos importantes

- **ORM (Object Relational Mapping)**  
  Permite trabalhar com banco de dados usando objetos em vez de SQL diretamente.

- **Classe → Tabela**

- **Objeto → Registro (linha da tabela)**

- **Atributos → Colunas**

---

## Funcionalidades

O projeto demonstra operações básicas:

- Criar produtos
- Listar produtos
- Atualizar informações
- Remover produtos
